WHAT'S NEW — v2.0
◈ Resize modes — Free & Locked replace streamer/personal mode. Toggle with R
◇ Scoring modes — Reform, Celestial, Signicial & Shoegazer via Ctrl 1–4
♬ LN Course mode — Experimental rank estimation for 4K LN Dan Courses

RESIZE MODE
R press to try
◱ Free — drag each edge independently ◈ Locked — scales everything up proportionally
Click the overlay window first if the key doesn't respond. To reset size, press Ctrl+R.
Current: Free
SCORING MODES
try the key combos below
Ctrl1 ◇ Reform default Ctrl2 ✦ Celestial Ctrl3 ◆ Signicial Ctrl4 ♫ Shoegazer
♬ LN Course mode activates automatically when a map has >45% LN notes
⚠ These modes are experimental — do not expect exact results
LAYOUT MODES
press L to cycle through layouts
◱ Full — complete info view, all metrics visible ◇ Simplified — dan name & score, minimal height ▪ Compact — thin bar, ideal for stream layouts
Layout is saved automatically between sessions
⚠ Layout modes are only available in the Modern skin
